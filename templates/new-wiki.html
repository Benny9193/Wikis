Create a New Wiki

Follow these steps to set up a new wiki in your hub

Create the Wiki Folder

Create a new folder inside the wikis/ directory with your wiki's name:

wikis/your-wiki-name/

Set Up the Folder Structure

Create the following structure inside your new wiki folder:

  • your-wiki-name/
    • index.html
    • page-1.html
    • page-2.html
    • images/

Copy the Template

Use the template below as a starting point for your wiki pages. Copy this HTML structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title - Your Wiki Name</title>
    <link rel="stylesheet" href="../../assets/css/wiki.css">
</head>
<body>
    <div class="wiki-layout">
        <aside class="wiki-sidebar">
            <!-- Sidebar content -->
        </aside>
        <main class="wiki-main">
            <article class="wiki-article">
                <!-- Your content -->
            </article>
        </main>
    </div>
</body>
</html>

Add to the Hub

Open index.html in the root folder and add a new wiki card to the grid:

<a href="wikis/your-wiki-name/index.html" class="wiki-card">
    <div class="wiki-icon">
        <!-- Choose an icon -->
    </div>
    <h2 class="wiki-title">Your Wiki Name</h2>
    <p class="wiki-description">Description of your wiki</p>
    <span class="wiki-meta">Category</span>
</a>

Start Writing!

Add your content using standard HTML. The wiki styles support:

  • Headings (h1-h4)
  • Paragraphs and lists
  • Code blocks and inline code
  • Tables and blockquotes
  • Info boxes (note, warning, tip)
  • Navigation between pages

Quick Start

Check out the existing wikis in the wikis/ folder for complete examples of wiki structure and styling.
